# public/

Static files served at the site root.

## Add your resume

Drop your resume here as **`resume.pdf`**. It will then be available at
`/resume.pdf` and every "Résumé" link on the site will work automatically.

```
public/resume.pdf
```

(Optional) Add an `og-image.png` here later for nice link previews when the
site is shared — see ROADMAP.md, Phase 6.
